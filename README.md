# Education_DA
анализ данных в образовании

## Программа курса:

1. Введение в анализ данных. Данные в образовании. Открытые платформы данных. (лекция 2 часа, практика 2 часа)
   - Анализ данных в современном мире. Его роль при построении систем ИИ.
   - Этапы анализа данных. Задачи анализа данных
   - Анализ данных в образовании. Цели и задачи.
   - Сбор данных. Открытые платформы данных.
   - Формат хранения и представления данных для анализа.
   - ПРАКТИКА: Анализ данных в среде Jupyter Notebook. Знакомство с табличными данными. Работа с пакетом Pandas.
3. Предподготовка даннных и исследовательский анализ данных. Визуализация образовательных данных. (лекция 2 часа, практика 4 часа)
  - Типы данных. Виды признаков.
  - Задачи предподготовки данных для последующего анализа: заполнение пропущенных данных, преобразование типов даннных, удаление дубликатов, некорректных данных.
  - Данные с образовательных порталов. Обеспечение защиты персональных данных. Обезличивание данных.
  - Исследовательский анализ данных. Задачи и методы.
  - ПРАКТИКА: Работа с данными из LMS Moodle. Экспорт данных. Загрузка и первичный анализ данных. Предобработка данных
  - ПРАКТИКА: Загрузка данных с открытых платформ (Kaggle, OpenICPSR, World Bank Education Statistics). Разведочный анализ данных.
  - СРС: Работа с открытыми наборами даннных. Наборы открытых данных Федеральной службы по надзору в сфере образования и науки. Наборы открытых данных Министерства науки и высшего образования
4. Статистичесский анализ данных (лекции 4 часа, практика 6 часов)
  - Модели зависимости двух величин. Анализ парных зависимостей.
  - Линейная зависимость. Понятие корреляции. Оценка силы линейной зависимости.
  - Коэффициенты корреляций: коэффициент Пирсона, коэффициент Спирмена, коэффициент Кендалла, условия применения каждого коэффициента.
  - Роль корреляционного анализа в выявлении зависимостей между образовательными показателями (успеваемости, активности и т.д.).
  - Примеры применения в образовании
  - ПРАКТИКА: Корреляционный анализ в Python. Работа с пакетами numpy, scipy. Расчет коэффициента Пирсона на основе образовательных данных. Интерпретация результатов и формулировка рекомендаций.
  - ПРАКТИКА: Исследование связи между активностью студентов в системе дистанционного обучения (LMS) и их итоговыми результатами на основе коэффициента Пирсона.
  - ПРАКТИКА: Исследование линейной связи между образовательными параметрами на основе коэффициентов Спирмена и Кендалла.
  - СРС: Исследование корреляции между уровнем вовлеченности в учебным процесс и уровнем результатов обучения. Анализ результатов, формулирование выводов.
5. Регрессионный анализ данных (лекции 4 часа, практика 6 часов)
  - Понятие регрессионного анализа. Построение регрессионной модели. 
  - Применение регрессионного анализа для решения образовательных задач: прогнозирование успеваемости, оценка эффективности обучения, выявление факторов успеха.
  - Простая линейная регрессия. Построение линейной регресии. Критерии оценки качества полученной модели.
  - Построение множественной линейной регрессии. Регуляризация моделей. L1, L2 - регуляризация. Выбор коэффициента регуляризации.
  - Добавление и анализ влияния признаков на качество модели. Добавление полиномиальных и бинарных признаков. Исследование и сравнение моделей.
  - Решение задачи классификации с помощью логистической регрессии. Построение логистической регрессии. Оценка точности классификатора.
  - Примеры использования: предсказание результатов тестирования, анализ влияния активности на итоговые оценки и т.д.
  - ПРАКТИКА: Построение простой линейной регрессии на основе образовательных данных. Построение множественной линейной регрессии. Оценка точности модели. Интерпретация полученных результатов
  - ПРАКТИКА: Построение полиномиальной регрессии на основе образовательных данных. Интерпретация полученных результатов
  - ПРАКТИКА: Построение классификатора: логистическая регрессия. Интерпретация полученных результатов
6. Снижение размерности. Факторный анализ. Анализ кластеров. (Лекция 2 часа. Практика 4 часа)
  - Факторный анализ данных. Задачи и методы. Метод главных коммпонентов. Анализ факторов.
  - Проецирование данных из многомерных пространств. Визуализация структуры многомерных данных.
  - Кластерный анализ данных. Метод k средних. Метод DBSCAN. Выявление аномалий в образовательных данных.
  - ПРАКТИКА: Применение метода главных компонент для выявления факторов успеха в образовательном процессе.
  - ПРАКТИКА: Кластеризация учающихся по их результатам обучения. Выявление проблемных зон в процессе обучения.
